# Integrations

## Data Sources
- **Local Data**: `src/data.js` exports a `DOMAINS` array containing all city, stay, and history data.
- **External Images**: Hosted on Unsplash (hotlinked in `data.js`).

## External Services
- **Stitch**: Source HTML files provided in `src/stitch-raw/`.
- **Navigation**: External links to hotel websites (e.g., mamounia.com).
