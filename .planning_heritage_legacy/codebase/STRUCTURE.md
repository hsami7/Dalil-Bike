# Structure

## Key Directories
- `src/`: Main source code.
- `src/assets/`: Static assets (SVG, PNG).
- `src/stitch-raw/`: Original HTML screen exports from Stitch.

## Key Files
- `src/main.jsx`: Application entry point.
- `src/App.jsx`: Main router and global layout.
- `src/data.js`: Centralized data store.
- `src/jsxify.js`: Utility for converting HTML to JSX (likely used during migration).
